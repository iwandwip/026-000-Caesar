# Firmware Notes

## Nextion Page ID Mapping

Page ID follows page order in Nextion Editor. Use these IDs when declaring `NexPage`, `NexButton`, `NexText`, `NexNumber`, and other Nextion components in Arduino code.

| Page ID | Page Name |
| --- | --- |
| 0 | pageDashboard |
| 1 | pageLoginF |
| 2 | pageLoginB |
| 3 | pageMldF |
| 4 | pageMldB |
| 5 | pageLotF |
| 6 | pageLotB |
| 7 | pageOutputNG |
| 8 | pageInputNGF |
| 9 | pageInputNGB |
| 10 | pageDtF |
| 11 | pageDtB |
| 12 | pageDtInfoF |
| 13 | pageDtInfoB |
| 14 | pageDtMc |
| 15 | pageDtMcInfo |
| 16 | pageSim |
| 17 | pageSys |
| 18 | keybdA |
| 19 | keybdB |

## Login Page IDs

`pageLoginF` uses Page ID `1`.

`pageLoginB` uses Page ID `2`.
